# my-app

Next.js + TypeScript + Tailwind + Sanity (embedded Studio).

## Getting started

1. Node
   - Use Node 20: `nvm use` (see `.nvmrc`)
2. Install deps
   - `npm install`
3. Environment
   - `cp .env.local.example .env.local` and set values
4. Dev
   - `npm run dev`
   - Studio at `/studio`, posts page at `/posts`
